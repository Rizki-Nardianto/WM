    TF-IDF (Term Frequency-Inverse Document Frequency)¶

    Tabah terakhir dari text pre-processing adalah term-weighting atau pembobotan term. Dimana term-weighting merupakan proses pemberian bobot pada setiap term dokumen. pembobotan ini diperlukan untuk nantinya diproses oleh algoritma Machine Learning dalam klasifikasi dokumen. Ada beberpa metode yang dapat digunakan dalam pembobotan, salah satunya yaitu TF-IDF (Term Frequency-Inverse Document Frequency).

    TF-IDF adalah ukuran statistik yang menggambarkan pentingnya suatu istilah terhadap sebuah dokumen dalam sebuah kumpulan atau korpus 1. ukuran ini sering digunakan sebagai faktor pembobot dalam pencarian temu balik informasi (information retrieval), penambangan text (text mining), dan pemodelan pengguna (user-modeling). Dalam TF-IDF Term frequency menyatakan fekuensi (tingkat keseringan) munculnya seuatu term dalam dokumen, sedangkan document frequency merupakan banyaknya jumlah dokumen dimana sebuah term itu muncul2.

    Rumus dari TF-ID adalah sebagai berikut: $$ W_{i,j}=tf_{i,j}\times \log ^{\cdot }\left( \dfrac{N}{df_{i}}\right) $$ Dimana tf menyatakan nilai dari Term frequency dan log (N/df) menyatakan nilai IDF dengan N adalah jumlah banyaknya data dalam satu dokumen.

    Proses TF-IDF¶

    Prepare Data¶

    Siapakan data yang akan diolah, baca file menggunakan Pandas. Kolom yang dibutuhkan dalam pemrosesan kali ini adalah kolom label dan tweet_tokens_stemmed . Kemudian rename kolom pada Dataframe menjadi ["label", "tweet"] .

    import pandas as pd 
import numpy as np

TWEET_DATA = pd.read_csv("Text_Preprocessing.csv", usecols=["label", "tweet_tokens_stemmed"])
TWEET_DATA.columns = ["label", "tweet"]

TWEET_DATA.head()

    Jika ketika kita cek, item di kolom tweet memiliki tipe data <class str> , maka kita perlu mengubah string menjadi list dan menyimpan pada Pandas Seriec tweet_list , 

    # convert list formated string to list
import ast

def convert_text_list(texts):
    texts = ast.literal_eval(texts)
    return [text for text in texts]

TWEET_DATA["tweet_list"] = TWEET_DATA["tweet"].apply(convert_text_list)

    Term Frequency (TF)¶

    Setelah me-load data diatas selanjutnya membuat TF (Term Frequency), dimana yang sudah dijelaskan diatas Term Frequency adlah frekuensi kemunculan term i pada dokumen j dibagi dengan total dari term pada documen j. kemudian dapat ditulis dalam bentuk,

    \[ \begin{aligned}tf_{ij}=\dfrac{f_{d}\left( i\right) }{\max f_{d}\left( j\right) }\\ j\in d\end{aligned} \]

    maka pada pemrograman Python dapat diimplementasikan dengan cara berikut,

    def calc_TF(document):
    # Counts the number of times the word appears in review
    TF_dict = {}
    for term in document:
        if term in TF_dict:
            TF_dict[term] += 1
        else:
            TF_dict[term] = 1
    # Computes tf for each word
    for term in TF_dict:
        TF_dict[term] = TF_dict[term] / len(document)
    return TF_dict

TWEET_DATA["TF_dict"] = TWEET_DATA['tweet_list'].apply(calc_TF)

TWEET_DATA["TF_dict"].head()

    document mrupakan Pandas Series row tweet_list yang sebelumnya dibuat. Kemudian itu .apply() fungsi .cacl_TF() kedalam Pandas Series tweet_list dan disimpan hasilnya pada TF_dict,

    # Check TF result
index = 90

print('%20s' % "term", "\t", "TF\n")
for key in TWEET_DATA["TF_dict"][index]:
    print('%20s' % key, "\t", TWEET_DATA["TF_dict"][index][key])

    Inverse Document Frequency (IDF)¶

    IDF berfungsi mengurangi bobot suatu term jika sering muncul dan tersebar diseruluh dokumen, kemudian IDF dapat dituliskan dalam persamaan

    \[ \begin{aligned}idf\left( t,D\right) =\log \dfrac{N}{|\{ d\in D:t\in d\}| }\\ \end{aligned} \]

    Dimana N menunjukan jumlah total dokumen dalam suatu corpus, N = |D|.

    |{d ∈ D : t ∈ d}| = df(t), merupakan jumlah dokumen yang mengandung term t.

    Dalam bentuk yang lain, IDF dapat dituliskan dengan persamaan,

    \[ idf\left( t,D\right) =\log ^{\cdot }\left( \dfrac{N}{df_{\left(t\right)} + 1}\right) \]

    Penambahan 1 untuk menghidarkan pembagian terhadap 0 jika df(t) tidak ditemukan di dalam corpus.

    Berikut merupakan implementasi pencarian df(t) pada bahasa perograman python,

    def calc_DF(tfDict):
    count_DF = {}
    # Run through each document's tf dictionary and increment countDict's (term, doc) pair
    for document in tfDict:
        for term in document:
            if term in count_DF:
                count_DF[term] += 1
            else:
                count_DF[term] = 1
    return count_DF

DF = calc_DF(TWEET_DATA["TF_dict"])

    Untuk menghitung nilai dari IDF pada pembrograman Python dapat menggunakan cara sabagai berikut,

    n_document = len(TWEET_DATA)

def calc_IDF(__n_document, __DF):
    IDF_Dict = {}
    for term in __DF:
        IDF_Dict[term] = np.log(__n_document / (__DF[term] + 1))
    return IDF_Dict

#Stores the idf dictionary
IDF = calc_IDF(n_document, DF)

    kemudian dari dictionary TF dan IDF kita bisa menemukan TF-IDF dengan mengkalikan keduanya dan disimpan dalam Pandas Series. Dalam pengerjaannya menggunakan .apply() terhadap fungsi calc_TF_IDF(), dalam implementasi python sebagai berikut,

    #calc TF-IDF
def calc_TF_IDF(TF):
    TF_IDF_Dict = {}
    #For each word in the review, we multiply its tf and its idf.
    for key in TF:
        TF_IDF_Dict[key] = TF[key] * IDF[key]
    return TF_IDF_Dict

#Stores the TF-IDF Series
TWEET_DATA["TF-IDF_dict"] = TWEET_DATA["TF_dict"].apply(calc_TF_IDF)

    hasil diatas (dictionary TF , IDF , dan Series TF-IDF ) kita dapat tampilkan dengan cara berikut,

    # Check TF-IDF result
index = 90

print('%20s' % "term", "\t", '%10s' % "TF", "\t", '%20s' % "TF-IDF\n")
for key in TWEET_DATA["TF-IDF_dict"][index]:
    print('%20s' % key, "\t", TWEET_DATA["TF_dict"][index][key] ,"\t" , TWEET_DATA["TF-IDF_dict"][index][key])

    Kemudian jadikan Series TF-IDF menjadi TF-IDF vector data berbentuk Sparse Matrix dengan dimensi (n_samples, n_feature), n_feature yang akan kita gunakan adalah top 50 term dengan TF-IDF terbesar.

    # sort descending by value for DF dictionary 
sorted_DF = sorted(DF.items(), key=lambda kv: kv[1], reverse=True)[:50]

# Create a list of unique words from sorted dictionay `sorted_DF`
unique_term = [item[0] for item in sorted_DF]

def calc_TF_IDF_Vec(__TF_IDF_Dict):
    TF_IDF_vector = [0.0] * len(unique_term)

    # For each unique word, if it is in the review, store its TF-IDF value.
    for i, term in enumerate(unique_term):
        if term in __TF_IDF_Dict:
            TF_IDF_vector[i] = __TF_IDF_Dict[term]
    return TF_IDF_vector

TWEET_DATA["TF_IDF_Vec"] = TWEET_DATA["TF-IDF_dict"].apply(calc_TF_IDF_Vec)

print("print first row matrix TF_IDF_Vec Series\n")
print(TWEET_DATA["TF_IDF_Vec"][0])

print("\nmatrix size : ", len(TWEET_DATA["TF_IDF_Vec"][0]))

    Menampilkan top 50 term TF-IDF dengan cara berikut,

    # Convert Series to List
TF_IDF_Vec_List = np.array(TWEET_DATA["TF_IDF_Vec"].to_list())

# Sum element vector in axis=0 
sums = TF_IDF_Vec_List.sum(axis=0)

data = []

for col, term in enumerate(unique_term):
    data.append((term, sums[col]))

ranking = pd.DataFrame(data, columns=['term', 'rank'])
ranking.sort_values('rank', ascending=False)
